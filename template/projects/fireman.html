Fireman Rescue

Fireman Rescue is a Serious Game which simulates different emergency situations for student firefighters of Institut de Seguretat PÃºblica de Catalunya (ISPC). It is made so students can practice their decision making on emergency situations without risks. Fireman Rescue was made with Unity using C# and developed at the company Viod Games while I was an Intern.

My Contribution

User Actions

Implemented the actions the user can do in the game, which depend on the situation. These were picking items from the truck, interacting with the NPCs in the scenario and using the equipped items during the simulation.

Objective System

Also implemented a simple objective system based on a Node Tree, which keeps track of the user actions and the available actions it can access from that point in the emergency. The objectives required for each level, were gathered from a Data Base as a JSON and then generated in-game. This allowed us to make changes for the level without having to change game files.

Score System

Following with the Objective System, I also worked on a Score System which rewards the player with more or less score depending on the order the actions were taken and also lowers the score if an action was not done during the simulation. This score was later used to show a rank obtained on the simulation so the students could improve and practice.

Fireman Helpers AI

My last contributuon was a very basic AI for Helpers, which were NPCs that the user can asign different actions to realize while the user focuses on other taks. They could open doors, put off fires, etc. This allowed the student to practice giving orders and splitting the work required on an emergency situation.
